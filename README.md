## Create python virtual environment

### Create virtual environment

Windows
```
python -m venv contraint-programming-env
```

MacOS
```
python3 -m venv contraint-programming-env
```

### Activate virtual environment

Windows
```
contraint-programming-env/Scripts/activate
```

MacOS
```
source contraint-programming-env/bin/activate
```
